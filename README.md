# ⚙️ Installation and Starting

Clone the repo and install dependencies:

```bash
git clone https://github.com/koalaporings/lettfaktura-terms.git
cd lettfaktura-terms
```

### Frontend
```
cd terms-fe
npm install
npm run dev
```

### Backend
```
cd terms-be
npm install
npm run dev
```
